# JsonApiConvention Module
[![Latest Stable Version](https://poser.pugx.org/spryker/json-api-convention/v/stable.svg)](https://packagist.org/packages/spryker/glue-json-api)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%207.2-8892BF.svg)](https://php.net/)

JsonApiConvetion module provides functionality to implement an ApiApplication using JSON API over HTTP

## Installation

```
composer require spryker/json-api-convention
```
